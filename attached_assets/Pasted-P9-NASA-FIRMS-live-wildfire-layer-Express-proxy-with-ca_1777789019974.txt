P9: NASA FIRMS live wildfire layer. Express proxy with caching, bundled MODIS CSV fallback.

Data source:
- URL: https://firms.modaps.eosdis.nasa.gov/api/area/csv/{FIRMS_API_KEY}/VIIRS_SNPP_NRT/world/1
- "world" = global bbox, "1" = last 24h
- VIIRS confidence field uses "l", "n", "h" strings (different from MODIS 0-100 numeric)

FIRMS_API_KEY is already in Replit Secrets (workspace + deployment), server-side only, no VITE_ prefix.

Bundled fallback: public/data/MODIS_C6_1_Global_24h.csv already in place.

Server side:

1. server/src/routes/fires.ts — Express GET /api/fires:
   - Read FIRMS_API_KEY from process.env
   - Fetch with 30-second timeout
   - Cache parsed JSON 30min in-memory
   - On upstream failure: fall back to bundled MODIS CSV
   - Return: { fires: [{lat, lon, confidence, frp, brightness, acq_date, acq_time, source}], source: "live"|"fallback"|"stale-cache" }
   - ALWAYS returns 200 with valid data

2. Register in server/src/index.ts.

Client side:

3. src/utils/api.ts — fetchFires() typed return.

4. src/layers/FiresLayer.ts:
   - Fetch on mount, refresh every 30 min
   - Single PointPrimitiveCollection (10k+ points)
   - pixelSize by FRP: <5MW=3, 5-50=5, >50=8
   - color by confidence: VIIRS l=yellow, n=orange, h=red. MODIS numeric: <30=yellow, 30-69=orange, ≥70=red
   - Subscribe to layerVisibility.fires, default false
   - Click handler: setSelectedEntity({type:'fire',id,data:{...}})

5. EntityPanel — handle type 'fire': coords, confidence (decoded l/n/h to Low/Nominal/High), FRP MW, brightness K, acq date/time, source.

6. LayerToggles fires row interactive, count = fires loaded.

7. California Wildfire Belt theater wires fires:true.

8. Confirm public/data/MODIS_C6_1_Global_24h.csv bundled into prod build.

Definition of done:
- /api/fires returns 10k-15k fires globally
- Toggle on → glow points dense over California, central Africa, Brazil, Indonesia, SE Asia
- California Wildfire Belt theater auto-enables fires
- Click point → entity panel with FRP/confidence
- Live API failure → MODIS fallback transparently

Do not touch theater, restricted airspace, trail rendering, or the tileset settings we just reverted.

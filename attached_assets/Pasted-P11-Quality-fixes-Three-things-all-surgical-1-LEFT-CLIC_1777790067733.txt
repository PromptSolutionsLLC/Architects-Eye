P11: Quality fixes. Three things, all surgical.

1. LEFT_CLICK handler clobber fix (architectural)

Currently every layer (Aircraft, Vessel, Satellite, Airspace, Fire) attaches its own LEFT_CLICK ScreenSpaceEventHandler. AircraftLayer's handler calls setSelectedEntity(null) on a miss, which can clobber a fresh selection from another layer if the aircraft handler fires last.

Fix: consolidate to ONE central LEFT_CLICK handler at the Viewer level. On click:
- viewer.scene.pick(movement.position) once
- Inspect picked.id or picked.primitive to determine entity type
- Dispatch setSelectedEntity({type, id, data}) based on what was picked
- If nothing picked → setSelectedEntity(null)

Remove the per-layer LEFT_CLICK handlers. Each layer keeps its own data/render pipeline, but the click resolution is centralized. Use a registry pattern: each layer registers a "pick resolver" function (input: picked object → output: SelectedEntity | null), Viewer iterates resolvers until one returns non-null.

Verify: click an aircraft, vessel, satellite, fire, airspace polygon → each populates the correct EntityPanel without flicker or clobber.

2. Vessel teleport detector

In VesselLayer when ingesting AISStream position updates: if the new position is >100 nautical miles from the previous position AND the time delta is <60 seconds, drop the update (treat as GPS glitch / MMSI collision). Log dropped updates to console as [VESSEL TELEPORT DROP] mmsi=X delta_nm=Y delta_s=Z. No store update for that tick.

3. Jamming hover tooltip

JammingLayer hexes currently have no hover affordance. Add a Cesium ScreenSpaceEventType.MOUSE_MOVE handler that picks the hex under the cursor and shows a small tooltip near the cursor:
- Hex H3 index
- Estimated jamming intensity (whatever value drives the red→orange gradient)
- "GPSJam.org · 2026-05-01"

Tooltip is a positioned absolute div, top-right offset from cursor, monospace font, dark background, 1px amber border. Hide on mouseout.

Definition of done:
- Click any entity type → correct EntityPanel, no clobber
- Vessel teleports no longer visible (drop count appears in console during 60s monitor)
- Jamming layer toggle ON → hover hex → tooltip appears with intensity + source

Do not touch P9 fires layer, theaters, trails, or tileset settings.

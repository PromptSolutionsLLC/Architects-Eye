P15: Cinematic polish. Five things, all visual. No new functionality.

1. FONTS — IBM Plex Mono / JetBrains Mono

Currently the UI uses default monospace (likely system mono or Courier). Replace with a proper terminal-grade monospace.

- Add Google Fonts import in index.html or via CSS @import:
  https://fonts.googleapis.com/css2?family=JetBrains+Mono:wght@400;500;700&family=IBM+Plex+Mono:wght@400;500;600&display=swap
- Use JetBrains Mono as primary, IBM Plex Mono fallback, monospace last resort
- Apply globally via CSS body/html font-family override
- Verify in HeaderBar, LayerToggles, EntityCards, SearchBox, Timeline, and the new boot screen all render in JetBrains Mono
- Numeric digits: prefer tabular figures (font-feature-settings: 'tnum') so counts in LayerToggles don't visually shift as values change

2. SCANLINE OVERLAY

Subtle CRT-scanline effect across the entire viewport. Sells the OSINT-terminal aesthetic hard.

- Full-screen <div> positioned fixed, top:0, left:0, width:100vw, height:100vh, pointer-events:none, z-index above globe but below all UI panels (z-index: 50 if panels are 100+)
- Background: repeating-linear-gradient(0deg, transparent 0px, transparent 1px, rgba(0,0,0,0.10) 2px, rgba(0,0,0,0.10) 3px)
- Opacity: 0.4 (subtle, not distracting)
- Hidden by default in dev; visible always in prod (or always — we want it on for the Loom regardless). Just ship it always-on.

3. FOG / ATMOSPHERE TUNING

Cesium has built-in atmosphere settings that go from "default flat" to "cinematic Earth." Tune these:

- viewer.scene.skyAtmosphere.show = true (currently false per earlier seam debug — re-enable now that seam is non-issue)
- viewer.scene.skyAtmosphere.hueShift = 0.0
- viewer.scene.skyAtmosphere.saturationShift = 0.1
- viewer.scene.skyAtmosphere.brightnessShift = 0.0
- viewer.scene.fog.enabled = true
- viewer.scene.fog.density = 0.0001 (subtle haze at altitude)
- viewer.scene.globe.showGroundAtmosphere = true
- viewer.scene.globe.atmosphereLightIntensity = 5.0
- viewer.scene.globe.atmosphereRayleighCoefficient = new Cesium.Cartesian3(5.5e-6, 13.0e-6, 28.4e-6)

These are well-documented Cesium tunings — agent: verify each property exists in the locked CesiumJS 1.140 version before applying. If any property name has changed in 1.140, use the equivalent.

4. BOOT TERMINAL LOADING SEQUENCE

Black screen with monospace amber text typing out, then fades to globe.

components/BootScreen.tsx — new file:

- Mounts at App root, covers entire viewport. position:fixed, inset:0, background: #000000, z-index: 9999
- Center-aligned content, monospace JetBrains Mono, amber #fbbf24
- Five lines, typed sequentially with character-level reveal:
  Line 1: "ARCHITECT'S EYE v1.0"
  Line 2: "INITIALIZING SECURE OSINT TERMINAL..."
  Line 3: "FETCHING TLE CATALOG... [15,304 OBJECTS]"
  Line 4: "ESTABLISHING AISSTREAM CONNECTION..."
  Line 5: "LOADING SUBMARINE CABLE TOPOLOGY..."
  Line 6: "[ READY ]"

- Typing speed: 30ms per character. Pause 200ms between lines.
- After Line 6 appears, hold 600ms.
- Then BootScreen fades out over 800ms (opacity 1 → 0), globe is visible underneath.
- After fade complete, BootScreen unmounts (visibility: hidden + display: none, or React conditional unmount).

- A blinking cursor "_" appears at the end of the currently-typing line.
- Use a simple useEffect + setTimeout chain. No external typing library.
- Boot screen runs ONCE per page load. Does not reappear on HMR.
- ESC key OR clicking anywhere on the boot screen → instant skip to fade-out (for impatient demos and judges who've seen it before).

- Boot screen appears immediately on App mount, BEFORE Cesium globe initializes. The globe initializes underneath while boot is showing — so when boot fades, globe is ready.

5. PERF KILL-SWITCH (was in original P15 plan, keep it)

Pressing 'P' key toggles a perf-mode that disables: jamming hexes (1222 primitives), submarine cables (711 polylines), satellite trails. Aircraft/vessels/satellites/fires/quakes/airspace stay on. Bottom-right indicator shows "PERF MODE" pill in amber when active. For low-end devices or stress moments during demo. ESC also exits perf mode.

- After "[ READY ]" → 600ms hold, 800ms fade to globe
- Globe reveal: full atmosphere visible, blue Earth glow at limb, fog softens distance, scanlines overlay subtly visible
- Hard refresh → boot screen reappears (one per session)
- Click anywhere on boot screen → instant skip

- Press 'P' → jamming/cables/sat-trails hide, "PERF MODE" indicator appears bottom-right
- Press 'P' again or ESC → exits perf mode, layers return
